Add and manage custom widgets on the WordPress admin dashboard.

== Description ==
Admin Dashboard Enhancer is a powerful plugin that allows you to add and manage custom widgets on your WordPress admin dashboard. Customize your admin experience with widgets displaying HTML, text, shortcodes, and more.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/admin-dashboard-enhancer` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Settings -> Dashboard Widgets screen to configure the plugin.

== Frequently Asked Questions ==

= How do I add a custom widget to my dashboard? =
After activating the plugin, navigate to Settings -> Dashboard Widgets and enter the content for your custom widget. Save your settings and the widget will appear on your admin dashboard.

= What types of content can I add to the custom widgets? =
You can add HTML, text, and shortcodes to your custom widgets.

= Can I style my widgets? =
Yes, you can add custom CSS to style your widgets.
